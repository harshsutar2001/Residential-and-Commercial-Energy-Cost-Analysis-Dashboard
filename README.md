# Residential-and-Commercial-Energy-Cost-Analysis-Dashboard

Project Overview

This project presents an interactive Power BI dashboard that analyzes energy costs for residential and commercial buildings.
The goal of the project is to explore how customer type, building size, number of occupants, and region influence energy expenses.

Using data visualization techniques, the dashboard provides insights into energy cost patterns, customer distribution, and cost efficiency metrics.

This project demonstrates skills in data analysis, dashboard development, and business insight generation.

Dataset Description

The dataset contains information about energy consumption and cost metrics for different customers.

Key Columns in the Dataset

customer_id – Unique identifier for each customer

customer_type – Residential or Commercial customer

regions – Geographic region of the customer

building_size_m2 – Building size in square meters

occupants – Number of people occupying the building

energy_cost_brl – Total energy cost in Brazilian Real (BRL)

Cost per square meter – Energy cost per square meter

Cost per occupant – Energy cost per occupant

People per square meter – Density of occupants

Categorize building size – Size category of the building

Tools & Technologies Used

Power BI – Dashboard creation and visualization

Excel – Initial data exploration

Data Visualization Techniques – KPI indicators and analytical charts

Dashboard Visualizations

The dashboard contains multiple visual elements to analyze energy cost behavior.

1️⃣ Sum of Occupants by Customer Type

Chart Type: Stacked Column Chart

Shows how the number of occupants is distributed between residential and commercial customers.

2️⃣ Sum of Occupants by Occupants

Chart Type: Clustered Bar Chart

Displays the distribution of total occupants across different records to understand population patterns.

3️⃣ Count of Cost (BRL) by Building Size

Chart Type: Line Chart

Shows how the number of energy cost records varies across different building sizes.

4️⃣ Sum of Cost per Square Meter by Region

Chart Type: Pie Chart

Illustrates the proportion of energy cost per square meter across regions.

This helps identify which regions have the highest energy cost intensity.

5️⃣ Building Size Category by Customer Type

Chart Type: Column Chart

Shows the count of categorized building sizes for residential vs commercial customers.

6️⃣ Cost per Occupant vs Energy Cost

Chart Type: Scatter Plot

Analyzes the relationship between:

Energy Cost (BRL)

Cost per Occupant

This helps identify high-cost outliers and efficiency patterns.

Key Performance Indicators (KPIs)

The dashboard includes KPI cards to highlight important metrics:

Highest Square Root Price

Number of Customers

Average Energy Cost

Total Energy Cost

These KPIs provide a quick summary of the dataset.

Interactive Filters

The dashboard includes slicers to enable dynamic exploration:

Region Filter

Building Size Filter

Users can filter the dashboard to analyze specific segments of the dataset.

Key Insights

Some insights that can be derived from the dashboard include:

Energy costs vary significantly between residential and commercial buildings.

Regions show different energy cost per square meter patterns.

Larger buildings often show higher total energy costs.

Cost per occupant varies depending on occupancy density and building size.

Use slicers to filter by region or building size.
